# 安装说明

## 1. 构建前准备

请先确保以下命令可用：

- `python`
- `python -m pip`
- `node`
- `npm`
- `rustc`
- `cargo`

推荐先执行：

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\build\check_env.ps1 -Profile bundle
```

## 2. 构建 Python sidecar

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\build\build_sidecar.ps1
```

## 3. 构建桌面安装包

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\build\build_desktop.ps1
```

## 4. 开发模式启动

### 启动 API

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\dev\start_api.ps1
```

说明：

- 脚本默认使用更稳定的非 `reload` 模式启动本地 `FastAPI`
- 如果要排查热重载相关问题，再显式改为调试模式

### 启动网页原型

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\dev\start_web.ps1
```

说明：

- 默认访问地址为 `http://127.0.0.1:4173/`
- 当仓库路径包含中文或其他非 `ASCII` 字符时，脚本会自动切到 `build + preview` 模式，避免 `Vite dev` 白屏
- 如需联合查看 API 与网页原型，可额外使用 `.\scripts\dev\start_all.ps1`

### 启动桌面端

```powershell
cd apps/desktop
npm install
npm run tauri:dev
```
