# 已知问题

## 当前环境已确认的阻塞项

- 当前机器尚未检测到 `rustc` 和 `cargo`
- 因此 `npm run tauri:build` 目前无法完成最终桌面包构建
- 当前机器尚未检测到 `ollama`
- 因此模型状态页会提示本地模型未就绪

## 当前版本功能边界

- B站解析为第一版，复杂视频、多分P、登录态内容仍可能失败
- 问答当前是本地检索增强版，不是完整 LLM 生成版
- OCR、Embedding、FAISS 目前仍以骨架接入为主，未做完整性能调优

## 打包相关说明

- sidecar 打包脚本已提供，但需本机具备 `PyInstaller`
- 桌面安装包打包脚本已提供，但需本机具备 `Rust/Cargo` 与 `npm`
